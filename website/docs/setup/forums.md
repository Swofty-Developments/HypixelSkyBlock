# Forums Website Setup

The forums website integrates with the API service to provide a web interface for players.

## Prerequisites

- API Service must be running
- Follow the setup at [HypixelForums Repository](https://github.com/Swofty-Developments/HypixelForums)

## Setup

1. Ensure your API service is running:
   ```bash
   java -jar ServiceAPI.jar
   ```

2. Clone the forums repository:
   ```bash
   git clone https://github.com/Swofty-Developments/HypixelForums
   ```

3. Follow the installation instructions in the HypixelForums README

## Configuration

The forums connect to your API service. Make sure the API endpoint is configured correctly in the forums configuration.

## Features

- Player profiles
- Forums and discussions
- Integration with in-game data
- Admin panel access
